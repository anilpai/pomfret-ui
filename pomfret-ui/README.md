## Install NodeJS:  

```sh
brew install node
```

## Install Yarn :

```sh
brew install yarn
```

## Project Setup:

```sh
cd pomfret-ui
```

## Install dependencies (only once):

### `yarn install`

## To start the project:

### `yarn start `

## To test the project:

### `yarn test`


Runs the app in the development mode.
Open http://localhost:3000 to view it in the browser.
